### Small django application with to-do/check list for personal use
#### Technology: Python-Django, HTML/Bootstrap, SQLite
#### Main features:
- User login
- Add/modify/remove tasks
- Daily tasks
- Add/modify/remove tasks
- Search your task/note
- Task status
- Task start-end date
